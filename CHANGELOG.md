# 更新日志

记录这个项目的每次更新。版本号用 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2026-06-04

### 新增
- 🔁 `--revert`：一键撤销最近一次整理（通过 `.organize_log.json` 操作日志还原）
- 🛡 `--keep PATTERN`：保留匹配通配符的文件不动（如 `--keep "*.tmp"`），可多次使用
- 🆕 `--version` 标志
- 📂 扩展支持的文件类型：电子书、字体、设计稿、种子、相机 RAW、Opus 音频等

### 改进
- 更新 LICENSE 中的版权人信息
- README 中加入新功能说明

## [1.0.0] - 2026-06-04

### 新增
- 按文件类型把文件夹自动分类（图片 / 文档 / 视频 / 音频 / 压缩包 / 安装包 / 代码 / 其他）
- 默认预演模式，安全不误操作；`--apply` 真正执行
- `--by-date`：按修改日期 (年-月) 分类
- 重名冲突时自动加 `_1`、`_2` 后缀，绝不覆盖
- 跳过隐藏文件和子文件夹
