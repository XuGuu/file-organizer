# 更新日志

记录这个项目的每次更新。版本号用 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.1] - 2026-06-05

### 修复
- 🐍 兼容性：加 `from __future__ import annotations`，类型注解延迟求值，支持 Python 3.7+（含老 Mac 自带的 3.8/3.9）
- 🛡 `--revert` 遇到损坏的操作日志时优雅报错，而不是抛出原始异常
- 🧹 `.gitignore` 忽略 `__pycache__`，更新过时的文档字符串

## [1.2.0] - 2026-06-05

### 新增
- 🌲 `--depth N`：可有限递归整理子文件夹，自动跳过已有分类文件夹避免循环
- 🧪 17 个单元测试覆盖核心逻辑（含 5 个 `--depth` 用例）
- 🤖 GitHub Actions CI：每次 push 自动在 Python 3.10/3.11/3.12 上跑测试
- 🤫 `-q` / `--quiet`：简洁模式，移动大量文件时不刷屏

### 改进
- 更友好的错误提示（路径不存在或传了文件而不是文件夹时）
- README 加测试徽章、`--depth`/`--quiet` 用法示例

## [1.1.0] - 2026-06-04

### 新增
- 🔁 `--revert`：一键撤销最近一次整理（通过 `.organize_log.json` 操作日志还原）
- 🛡 `--keep PATTERN`：保留匹配通配符的文件不动（如 `--keep "*.tmp"`），可多次使用
- 🆕 `--version` 标志
- 📂 扩展支持的文件类型：电子书、字体、设计稿、种子、相机 RAW、Opus 音频等

### 改进
- 更新 LICENSE 中的版权人信息
- README 中加入新功能说明

## [1.0.0] - 2026-06-04

### 新增
- 按文件类型把文件夹自动分类（图片 / 文档 / 视频 / 音频 / 压缩包 / 安装包 / 代码 / 其他）
- 默认预演模式，安全不误操作；`--apply` 真正执行
- `--by-date`：按修改日期 (年-月) 分类
- 重名冲突时自动加 `_1`、`_2` 后缀，绝不覆盖
- 跳过隐藏文件和子文件夹
